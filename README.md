# Sprint_4

1. Автотест был подготовлен для тестирования сервиса "Яндекс Самокат".
2. Данный автотест проверяет следующее:
   - Заказ самоката;
   - Кликабельность логотипов "Яндекс" и "Самокат" и переход в нужную страницу.
   - Кликабельность вопросов в блоке "Важные вопросы".
3. Тест-кейсы в файле test_order_page:
   tc-1. Оформление заказа через кнопку расположенную сверху.
   tc-2. Переход на форму заказа через кнопку расположенную снизу.
   tc-3. Проверка кликабельности логотипа 'Самокат' и перехода на нужный адрес.
   tc-4. Проверка кликабельности логотипа 'Яндекс' и перехода на нужный адрес.
4. При создании тестов использовались следующие библиотеки: pytest, selenium, allure и другие. Более подробно можно посмотреть в requirements.txt.
5. Установить зависимости — pip install -r requirements.txt.
6. Команда для запуска — python -m pytest. 
7. Команда для запуска конкретного теста python -m pytest -s tests/test_order_page.py -k test_click_on_button_order_main
